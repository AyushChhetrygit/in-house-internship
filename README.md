# Customer Data Analytics & ML Pipeline

This project is a comprehensive end-to-end data analytics system designed to generate synthetic e-commerce user activity, perform predictive machine learning tasks (such as segmentation and churn prediction), and visualize the results in a full-stack dashboard.

## 🚀 Features

- **Synthetic Data Generation:** Generates highly realistic e-commerce customer datasets mimicking Amazon-like user behavior (account age, last purchase, orders, spend, wishlist activity, etc.).
- **Machine Learning Pipeline:** 
  - **Customer Segmentation:** Groups users into behavioral segments using K-Means Clustering on recency, frequency, and monetary values (RFM).
  - **Churn Risk Prediction:** Assesses the likelihood of users leaving the platform based on recent activity.
  - **Customer Lifetime Value (CLV):** Calculates projected lifetime spending.
  - **Decision Engine:** Recommends automated marketing actions based on the user's segment, risk, and CLV (e.g., "Retention Campaign", "Loyalty Program").
- **AWS S3 Integration:** Capable of continuously recording and streaming data into Amazon S3 storage buckets for batch or real-time analytics.
- **REST API Backend:** Node.js Express server to serve processed machine learning datasets as a REST API. 
- **Modern React Dashboard:** A responsive, interactive frontend built with Vite, React, Tailwind CSS, and Recharts to visualize KPIs and analytical metrics in real-time.

## 🛠️ Tech Stack

### Data Science & Machine Learning
- **Python 3.12+**
- **Pandas, NumPy:** Data manipulation and engineered features.
- **Scikit-Learn:** Unsupervised Machine Learning (K-Means Clustering) and data scaling.
- **Jupyter Notebook:** Job scheduling and continuous batch generation.
- **Boto3:** AWS SDK for uploading data to cloud storage.

### Backend Development
- **Node.js & Express.js:** Fast and minimalist backend framework.
- **csv-parser & Chokidar:** Parses large tabular datasets and watches for hot-reloads when the ML model updates the inference data.
- **Dotenv, Cors:** Configuration and routing.

### Frontend Development
- **React.js (v18) & Vite:** Ultra-fast, component-based view rendering.
- **Tailwind CSS:** Utility-first CSS styling for modern UIs.
- **Recharts:** Composable charting library to render beautiful graphs.
- **Framer Motion & Lucide React:** Smooth web animations and modern interface icons.

## 🏃 Getting Started

To install, configure, and run this application, refer to the detailed execution guide inside `run_commands.txt`.

In brief:
1. Install Python dependencies and run `synthetic_data_generator.py` followed by `ml_pipeline.py`.
2. Install standard Node modules (`npm install`) inside both `./client` and `./server`.
3. Start the Node.js API with `npm run dev` in the `./server` folder.
4. Launch the React interface with `npm run dev` in the `./client` folder.
